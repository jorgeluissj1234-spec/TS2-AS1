# Sitio Web Interactivo - Práctica de Programación Web

Este repositorio contiene un proyecto web interactivo básico desarrollado con HTML5, CSS3 y JavaScript (Vanilla JS). El objetivo principal es demostrar la integración de estructuras semánticas, estilos responsivos y manipulación básica del DOM mediante una interfaz interactiva y dinámica.

# 🚀 Características Principales

Estructura Semántica: Maquetación limpia utilizando etiquetas nativas de HTML5 (<header>, <section>, <nav>, <footer>).

Carrusel Dinámico de Imágenes: Componente con transiciones suaves utilizando clases de CSS y opacidad, configurado con soporte para autoplay automático e interrupción por interacción del usuario.

Manipulación del DOM en Tiempo Real:

Cambio interactivo de mensajes de texto mediante arreglos dinámicos en JavaScript.

Modificación de estados y escalado de elementos (efecto expandir botón).

Manejo de alertas personalizadas en enlaces de navegación.

Diseño Visual: Estilos fluidos creados con CSS moderno (Flexbox, gradientes, transiciones de estado) e integración de iconos vectoriales mediante Font Awesome.

Sección Oculta (Easter Egg): Inclusión de un componente visual interactivo oculto en el pie de página con efecto hover avanzado.

# 🛠️ Tecnologías Utilizadas

HTML5 - Estructuración del contenido y elementos multimedia.

CSS3 - Animaciones, transiciones, diseño de contenedores (aspect-ratio, positioning) y layouts flexibles.

JavaScript (ES6) - Lógica del carrusel, eventos de clic, manejo de temporizadores (setInterval) y control de clases dinámicas.

# Estructura del Código

El proyecto se compone de tres archivos principales:

index.html: Define la estructura del sitio, organizando las secciones de Hero, Galería, Interacción y Footer.

styles.css: Contiene las reglas estéticas, efectos de transición para el carrusel y las propiedades visuales de los estados generales (hover, active).

script.js: Controla el flujo interactivo del sitio, incluyendo el ciclo automático del carrusel, los arreglos de mensajes mutables y la captura de eventos en los botones.
